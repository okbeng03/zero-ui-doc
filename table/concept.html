# JSON schema

因为列表一般展示的是「表单」保存进库的数据，表单中 JSON Schema 很好地描述了 JSON 数据结构，部分描述我们希望能复用到列表中：

  • 数据 formatter: 通过 schema 描述实现一些默认 customRender，以实现更友好的展示
  • 生成查询表单
