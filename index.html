# zero-ui

基于 vue.js 的前端中后台配置化UI库。包含「表单」和「表格」的配置化生成。

# 特性

  • 基于ant-design-vue，统一化UI
  • 配置化生成 Form、Table，更少的编码
    • 基于JSON Schema JSON 数据结构规范描述
    • 辅以「definition」扩展配置化描述
  • 支持扩展「模块插件」、「自定义规则组件」

# 依赖

  • vue.js
  • ant-design-vue
  • lodash
  • ajv
  • moment
